# RBR SRP Rally Creator
 A handy graphical tool for creation of RBR rallies compatible with WorkerBee's TrainingDay plugin.
 
# Current state of the project
 At the moment, the app is able to read, parse, write, translate and locate SRP files in RBR's directory. Read TrackListing.CSV and generate a list of accepted combinations of stage/weather/conditions/et cetera.<br/>
